## 既知の不具合

* ROM横のLEDの表示が反転しています。ROMが選択されていないときは点灯し、選択されているときは消灯します。
  * LEDにつながっているGNDを5Vに置き換えることで修正可能だと思われます。上の部品リストには含んでいません。
